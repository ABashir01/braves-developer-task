Instructions for running:
1. Open a terminal
2. Navigate to the 'backend' folder
3. Run the 'server.py' file in that terminal however you'd like (writing 'py server.py', etc.) 
4. If you have issues running server.py, make sure you have pip installed the necessary dependencies (flask, flask_cors, sqlite3)
5. Open a second terminal while server.py is running in the first
6. Navigate to the 'frontend' folder
7. Write 'npm install' to install the necessary dependencies
8. Write 'npm run start'
9. The app should open in your browser
10. Use it by writing a player into the search bar!
